Scripting Reference

Alle basis commando's, concepten en voorbeelden op één plek. Geen volledige scripts — gewoon de bouwstenen.

Bash

Basics & shebang

Elk bash script begint met een shebang en heeft uitvoerrechten nodig.

Shebang

Vertelt het systeem welke interpreter het script moet uitvoeren. Moet op de allereerste regel staan, zonder spaties.

#!/bin/bash
# Dit is commentaar
echo "Hallo wereld"

Script uitvoeren

Uitvoerrechten geven en daarna starten.

chmod +x script.sh        # uitvoerrechten geven
./script.sh               # script uitvoeren
bash script.sh            # ook geldig, zonder chmod
bash -x script.sh         # debug modus
CommandoUitleg
chmod +x bestandGeeft uitvoerrechten aan een bestand
which bashToont pad naar de bash interpreter
bash --versionToont versie van bash
cat /etc/shellsLijst van beschikbare shells op het systeem
Bash

Variabelen

Variabelen toekennen zonder $, uitlezen mét $. Geen spaties rond het = teken.

Declareren en gebruiken

naam="Gunther"
leeftijd=20
pad="/etc"

echo "Hallo $naam"
echo "Je bent ${leeftijd} jaar oud"   # accolades voor duidelijkheid

Command substitutie

Uitvoer van een commando opslaan in een variabele met $().

gebruiker=$(whoami)
datum=$(date)
bestanden=$(ls /etc)

echo "Ingelogd als: $gebruiker"

Argumenten

Waarden meegeven bij het uitvoeren van een script.

# ./script.sh Alice 25
echo "Naam: $1"        # eerste argument
echo "Leeftijd: $2"   # tweede argument
echo "Aantal args: $#" # aantal argumenten
echo "Alle args: $@"  # alle argumenten
Let op: Gebruik altijd quotes rond variabelen: "$var" in plaats van $var. Zo voorkom je problemen met spaties in waarden.

Output tonen

Conditionals

Beslissingen nemen met if, elif, else en case.

if / elif / else

if [ "$naam" = "Gunther" ]; then
    echo "Hallo Gunther!"
elif [ "$naam" = "admin" ]; then
    echo "Beheerder aangemeld"
else
    echo "Onbekende gebruiker"
fi

case statement

Handig als alternatief voor veel elif-blokken.

    1) echo "Je koos optie 1" ;;

    *) echo "Ongeldige keuze" ;;
esac
Bash

Loops

Herhaling met for, while en until.

for loop

# Over een lijst
for fruit in appel peer banaan; do
    echo "Fruit: $fruit"
done

# Met een teller (C-stijl)
for (( i=1; i<=5; i++ )); do
    echo "Ronde $i"
done

# Over bestanden
for bestand in /etc/*.conf; do
    echo $bestand
done

while loop

teller=0
while [ $teller -lt 5 ]; do
    echo "Teller: $teller"
    (( teller++ ))
done

# Oneindige loop met break
while true; do
    read -p "Doorgaan? (j/n) " antw
    [ "$antw" = "n" ] && break
done
KeywordUitleg
breakVerlaat de loop onmiddellijk
continueSla de rest van deze iteratie over
Bash

Functies

Herbruikbare blokken code. Argumenten worden doorgegeven via $1, $2...

Functie definiëren en aanroepen

begroet() {
    echo "Hallo, $1!"
}

begroet "Gunther"   # aanroepen
begroet "wereld"

Returnwaarde

Bash functies geven geen waarden terug zoals in andere talen. Gebruik echo + command substitutie.

kwadraat() {
    echo $(( $1 * $1 ))
}

result=$(kwadraat 5)
echo "5² = $result"   # 5² = 25

Pipes & redirectie

Uitvoer doorgeven, opslaan of weggooien.

Pipes

Uitvoer van één commando doorgeven aan het volgende met |.

ls /etc | grep "conf"           # filter op .conf bestanden
cat bestand.txt | wc -l        # tel regels
df -h / | tail -n 1            # enkel de data-rij
ps aux | grep "nginx"         # zoek lopend proces

Redirectie

echo "tekst" > bestand.txt      # overschrijven
echo "meer" >> bestand.txt      # toevoegen (append)
commando 2> fout.log            # foutmeldingen opslaan
commando > /dev/null 2>&1      # alle output weggooien
OperatorUitleg
>Stdout naar bestand (overschrijft)
>>Stdout naar bestand (voegt toe)
2>Stderr naar bestand
2>&1Stderr samenvoegen met stdout
<Bestand als stdin gebruiken
|Stdout van links naar stdin van rechts
Bash

Strings & rekenwerk

Rekenwerk

Gebruik $(( )) voor gehele getallen.

a=10; b=3
echo $(( a + b ))     # 13
echo $(( a * b ))     # 30
echo $(( a / b ))     # 3  (enkel gehele deling)
echo $(( a % b ))     # 1  (rest)
(( a++ ))              # increment

Stringmanipulatie

tekst="Hallo Wereld"
echo ${#tekst}          # lengte: 12
echo ${tekst,,}         # kleine letters
echo ${tekst^^}         # hoofdletters
echo ${tekst:0:5}       # substring: "Hallo"
echo ${tekst/Hallo/Dag} # vervangen
Bash

Exit codes

Elk commando geeft een exitcode terug. 0 = succes, alles anders = fout.

ls /bestaat-niet
echo $?           # toont exitcode van vorige commando

if ping -c 1 8.8.8.8 >/dev/null 2>&1; then
    echo "Netwerk OK"
else
    echo "Geen verbinding"
fi

exit 0    # script eindigt met succes
exit 1    # script eindigt met fout
OperatorUitleg
cmd1 && cmd2cmd2 alleen uitvoeren als cmd1 slaagde
cmd1 || cmd2cmd2 alleen uitvoeren als cmd1 faalde
$?Exitcode van het laatste commando
PowerShell

Basics & uitvoeren

PowerShell scripts hebben de extensie .ps1. Op Windows moet de execution policy soms worden aangepast.

Script uitvoeren

# Execution policy instellen (eenmalig, als admin)
Set-ExecutionPolicy RemoteSigned

# Script uitvoeren
.\script.ps1

# Of direct via PowerShell
powershell -File script.ps1

Handige basis-cmdlets

CmdletUitleg
Get-Help cmdletHulp tonen over een cmdlet
Get-Command *tekst*Cmdlets zoeken op naam
Clear-HostTerminal leegmaken
$PSVersionTablePowerShell versie tonen
Get-LocationHuidig pad (= pwd)
Set-Location C:\Navigeren (= cd)
Tip: PowerShell heeft aliassen voor veel Unix-commando's. ls, cd, cat, echo werken ook in PowerShell.
PowerShell

Variabelen & types

Variabelen beginnen altijd met $, zowel bij toekennen als bij uitlezen.

Declareren en types

$naam = "Gunther"
$leeftijd = 20
$prijs = 9.99
$actief = $true        # boolean
$lijst = @(1, 2, 3)   # array

# Type afdwingen
[int]$getal = "42"
[string]$tekst = 100

Strings

Dubbele quotes laten variabelen toe; enkele quotes niet.

$naam = "Gunther"
Write-Host "Hallo $naam"      # Hallo Gunther
Write-Host 'Hallo $naam'      # Hallo $naam (letterlijk)

# String interpolatie met expressies
Write-Host "2 + 2 = $(2+2)"  # 2 + 2 = 4

Arrays & hashtables

$kleuren = @("rood", "groen", "blauw")
Write-Host $kleuren[0]       # rood
Write-Host $kleuren.Count     # 3

$persoon = @{
    Naam = "Gunther"
    Leeftijd = 20
}
Write-Host $persoon.Naam      # Gunther

Pipeline & filtering

De pipeline in PowerShell werkt met objecten, niet met platte tekst.

# Filteren met Where-Object
Get-Process | Where-Object { $_.CPU -gt 10 }

# Sorteren
Get-Service | Sort-Object Name

# Selecteer kolommen
Get-Process | Select-Object Name, CPU, Id

# Eerste/laatste N items
Get-ChildItem | Select-Object -First 5

# Tellen
Get-Service | Measure-Object
PowerShell

Objecten & properties

PowerShell werkt met .NET objecten. Elke output heeft properties en methods.

# Alle properties bekijken
Get-Process notepad | Get-Member

# Specifieke property opvragen
$proc = Get-Process notepad
Write-Host $proc.Id
Write-Host $proc.Name
Write-Host $proc.CPU
PowerShell

Foutafhandeling

try / catch / finally

try {
    Get-Content "bestaat-niet.txt" -ErrorAction Stop
    Write-Host "Bestand gelezen"
}
catch {
    Write-Host "Fout: $($_.Exception.Message)"
}
finally {
    Write-Host "Dit wordt altijd uitgevoerd"
}
ParameterUitleg
-ErrorAction StopFout wordt een exception (nodig voor try/catch)
-ErrorAction SilentlyContinueFout negeren, doorgaan
$Error[0]Laatste fout opvragen
$?True/False: was het laatste commando succesvol?

Bash vs PowerShell

Dezelfde taak, twee talen. Handig als referentie.

Bash

# Variabele
naam="Gunther"
echo "$naam"

PowerShell

# Variabele
$naam = "Gunther"
Write-Host $naam

Bash

# if statement
if [ $x -gt 10 ]; then
    echo "groot"
fi

PowerShell

# if statement
if ($x -gt 10) {
    Write-Host "groot"
}

Bash

# for loop
for i in 1 2 3; do
    echo $i
done

PowerShell

# for loop
foreach ($i in 1..3) {
    Write-Host $i
}

Bash

# Bestand lezen
cat bestand.txt

# Filteren
cat log.txt | grep "fout"

PowerShell

# Bestand lezen
Get-Content bestand.txt

# Filteren
Get-Content log.txt | Select-String "fout"

Bash

# Functie
groet() {
    echo "Hallo $1"
}
groet "Wereld"

PowerShell

# Functie
function Groet($naam) {
    Write-Host "Hallo $naam"
}
Groet "Wereld"
