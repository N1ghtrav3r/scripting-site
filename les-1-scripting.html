Shells

Op één systeem kunnen meerdere shells aanwezig zijn. In Linux is bash meestal de populairste shell; op macOS is vaak zsh standaard, en op Windows heb je command shell en PowerShell.

cat /etc/shells | xargs ls -go
chsh
env

De standaard login shell van een gebruiker staat in /etc/passwd. Met chsh kan je die wijzigen. fileciteturn2file0L5-L6

Wat is een bash script?

  • Een bash script is een plat tekstbestand.
  • Het bevat een reeks commando’s.
  • Veel van die commando’s kan je ook gewoon rechtstreeks in de commandline typen.
  • Het hoeft niet gecompileerd te worden.
  • De extensie .sh is gebruikelijk, maar niet verplicht.

De slides benadrukken expliciet dat alles wat je op de commandline kan doen, ook in een script kan, en omgekeerd. fileciteturn2file0L11-L11

Hoe werkt dat intern?

Wanneer je in de terminal werkt, draait er al een bash-proces. Start je een script, dan wordt dat normaal uitgevoerd in een nieuw proces. Daardoor heeft een script vaak zijn eigen context.

De cursus gebruikt het voorbeeld van twee aparte cp-processen als je hetzelfde commando in twee terminals start. fileciteturn2file0L12-L12

Script uitvoeren

chmod +x myscript.sh
./myscript.sh
bash myscript.sh
bash -x myscript.sh

Voor je een script rechtstreeks uitvoert, moet het uitvoerrechten hebben. Vergeet je dat, dan krijg je een fout zoals Permission denied. Met bash -x kan je debuggen. fileciteturn2file0L13-L16

Shebang

#!/bin/bash
# commentaar
echo Hello students!
  • De shebang moet op de allereerste regel staan.
  • Geen spaties voor # of tussen ! en het pad.
  • Gebruik best een absoluut pad.

Variabelen

Met variabelen doe je in bash twee dingen: een waarde toekennen en die waarde later uitlezen.

myvariable=hello
anothervar=Fred
echo $myvariable $anothervar

sampledir=/etc
ls $sampledir
  • Bij instellen gebruik je géén $.
  • Bij uitlezen gebruik je wel $.
  • Geen spaties rond het gelijkteken.
  • Bash is hoofdlettergevoelig.

Command line arguments worden aangesproken via $1, $2, enzovoort. fileciteturn2file0L19-L22
