Integers

Een integer is een geheel getal. In bash kan je dat expliciet declareren met declare -i.

#!/bin/bash
declare -i y=10
declare -i getal=0
declare -i som=0

read getal
som=$((y + getal))
echo $som

Belangrijk: voor rekenen gebruik je meestal $(( ... )).

If / elif / else

Met if test je een conditie. Is die waar, dan voer je de code uit. Anders ga je naar else of een volgende elif.

Eenvoudige if/else

#!/bin/bash
if [ $1 -eq $2 ]; then
  echo "they are equal"
else
  echo "they are NOT equal"
fi

Meerdere voorwaarden met elif

#!/bin/bash
if [ $1 -eq $2 ]; then
  echo "they are equal"
elif [ $1 -gt $2 ]; then
  echo "first is greater than second"
elif [ $1 -lt $2 ]; then
  echo "first is smaller than second"
fi

Hoe lees je dit?

  • if = eerste test
  • then = wat doen als test waar is
  • elif = anders, test volgende voorwaarde
  • else = niets van hierboven was waar
  • fi = einde van het if-blok

test, [ ] , [[ ]] en (( ))

[ ] is hetzelfde als test

test -f bestand.txt
[ -f bestand.txt ]

Die twee doen hetzelfde. Spaties zijn hier heel belangrijk, omdat [ als een commando wordt gelezen.

[[ ]] is moderner

read naam
if [[ $naam = *A* ]]; then
  echo "welcome $naam, containing A"
fi

[[ ]] is veiliger en handiger voor strings en pattern matching.

(( )) is voor getallen

n=1
while (( n <= 5 )); do
  echo "welcome $n"
  n=$((n+1))
done

Voor numerieke vergelijkingen is (( )) meestal leesbaarder dan de oude stijl met -le, -gt, enzovoort.

Praktische regel

  • [ ] = klassiek en compatibel
  • [[ ]] = beter voor strings en files
  • (( )) = best voor getallen

Modernere numerieke stijl

if (( a > b )); then

String vergelijkingen

if [ "$naam" = "Gunther" ]; then
  echo "Welkom"

if [ "$naam" != "Gunther" ]; then

if [[ $naam = *A* ]]; then

Case

case gebruik je wanneer je verschillende mogelijke waarden wilt afhandelen. Dat is vaak duidelijker dan een lange rij if/elif.

#!/bin/bash
day=$(date +"%a")

case $day in
  Mon|Tue|Wed|Thu|Fri)
    echo "today is a weekday"
    ;;
  Sat|Sun)
    echo "today is the weekend"
    ;;
  *)
    echo "date not recognized"
    ;;
esac

Belangrijk

  • case $variabele in start het blok
  • Elke optie eindigt met ;;
  • * is de standaardoptie
  • Het blok eindigt met esac

Loops

Met loops laat je bash meerdere keren dezelfde code uitvoeren.

for

Gebruik je als het aantal herhalingen of items vooraf gekend is.

while

Loopt zolang een conditie waar is.

until

Loopt totdat een conditie waar wordt. Eigenlijk omgekeerd van while.

For

Over vaste lijst

for i in 1 2 3 4 5
do
  echo "Welcome $i times"
done

Met range

for i in {1..5}
do
  echo "Welcome $i times"
done

Met stappen

for i in {0..10..2}
do
  echo "Welcome $i times"
done

Stapwaarden werken in Bash 4.0 of nieuwer.

Over strings

for car in bmw ford toyota nissan
do
  echo "Value of car is: $car"
done

Over command output

for i in $(ls /tmp/*)
do
  echo "File $i"
done

Werkt, maar wees hier voorzichtig mee bij bestandsnamen met spaties.

While / Until

While

declare -i i=0
while (( i <= 4 ))
do
  echo $i
  ((i++))
done

while blijft lopen zolang de voorwaarde waar is.

Until

declare -i i=0
until (( i > 5 ))
do
  echo $i
  ((i++))
done

until blijft lopen zolang de voorwaarde nog niet waar is.

Oefeningen / voorbeelden

wiebenik.sh

Doel: kijk of naam.txt bestaat. Zo ja, lees de naam. Zo nee, vraag een naam en sla die op.

#!/bin/bash
if [ -f naam.txt ]; then
  naam=$(cat naam.txt)
  echo "U bent $naam"
else
  read -p "Wat is uw naam? " naam
  echo "$naam" > naam.txt
  echo "Bedankt. Uw naam is opgeslagen."
fi

num2txt.sh met case

Doel: zet één cijfer om naar tekst.

#!/bin/bash
case $1 in
  0) echo "nul" ;;
  1) echo "een" ;;
  2) echo "twee" ;;
  3) echo "drie" ;;
  4) echo "vier" ;;
  5) echo "vijf" ;;
  6) echo "zes" ;;
  7) echo "zeven" ;;
  8) echo "acht" ;;
  9) echo "negen" ;;
  *) echo "Ongeldige parameter $1" ;;
esac

Meerdere cijfers met for

#!/bin/bash
for (( i=0; i<${#1}; i++ )); do
  digit=${1:i:1}
  case $digit in
    0) echo "nul" ;;
    1) echo "een" ;;
    2) echo "twee" ;;
    3) echo "drie" ;;
    4) echo "vier" ;;
    5) echo "vijf" ;;
    6) echo "zes" ;;
    7) echo "zeven" ;;
    8) echo "acht" ;;
    9) echo "negen" ;;
    *) echo "Ongeldige parameter $digit" ;;
  esac
done
