Basics PowerShell 4

Wat is PowerShell?

PowerShell is een command line shell en scripting taal van Microsoft. In tegenstelling tot Bash werkt PowerShell met objecten in plaats van pure tekst.

Variabelen

$naam = "Gunther"
$leeftijd = 30
Write-Output "Hallo $naam"

Input vragen

$naam = Read-Host "Wat is je naam"
Write-Output "Welkom $naam"

Rekenen

$a = 5
$b = 3
$c = $a + $b
Write-Output $c

If statement

$getal = 10

if ($getal -gt 5) {
 Write-Output "Groter dan 5"
}

If else

if ($getal -gt 5) {
 Write-Output "Groter"
}
else {
 Write-Output "Kleiner"
}

Vergelijkingsoperatoren

  • -eq = gelijk
  • -ne = niet gelijk
  • -gt = groter dan
  • -lt = kleiner dan
  • -ge = groter of gelijk
  • -le = kleiner of gelijk

For loop

for ($i=1; $i -le 5; $i++) {
 Write-Output "Nummer $i"
}

Foreach

$lijst = 1,2,3,4
foreach ($item in $lijst) {
 Write-Output $item
}

While loop

$i = 1
while ($i -le 5) {
 Write-Output $i
$i++
}

Functie maken

function TelOp {
 param($a,$b)
 return $a + $b
}

$result = TelOp 5 10
Write-Output $result
