WhatIf, Confirm en Test-Connection

Get-Process Code | Stop-Process -Confirm
Test-Connection www.google.be

  • -Confirm vraagt bevestiging voor uitvoering.
  • Test-Connection is het PowerShell-alternatief voor ping.

Alles is een object

PowerShell werkt met objecten, niet gewoon met losse tekst.

  • Elk commando genereert een output van objecten.
  • Die output wordt in het geheugen opgebouwd.
  • Je kan objecten via de pipeline doorgeven, tonen, selecteren, filteren, sorteren en groeperen.
TermBetekenis
CollectionVerzameling objecten / tabel
ObjectEén element / rij
PropertyData-element / kolom
MethodActie die je met een object kan doen

Deze terminologie wordt letterlijk in de les uitgelegd. fileciteturn2file4L17-L22

Get-Member

Get-Process | Get-Member
Get-Service | Get-Member
  • Toont het type van het object.
  • Toont properties, methods en events.
  • Het type hangt af van het laatste commando vóór Get-Member.
  • Sommige commando’s geven geen output, dus dan werkt Get-Member niet.

De slides waarschuwen ook dat Get-Member vaak méér properties toont dan je normaal op het scherm ziet. fileciteturn2file4L23-L26

Sort-Object, Select-Object en Where-Object

Sort-Object

Get-Process | Sort-Object CPU -Descending
Get-Process | Sort-Object CPU, ProcessName

Sorteren verandert het objecttype niet. Je kan op meerdere properties sorteren. fileciteturn2file4L27-L28

Select-Object

Get-Process | Select-Object Name, Id, CPU
Get-Process | Sort-Object CPU -Descending | Select-Object Name, Id, CPU

Select-Object maakt nieuwe outputobjecten aan en wordt gebruikt om enkel de properties te behouden die je wil tonen of verder gebruiken. fileciteturn2file4L29-L31

Where-Object

Get-Service | Where-Object { $_.Status -eq "Running" }
Get-Process | Where-Object { $_.Name -like "*shell*" }

$_ verwijst naar het huidige object in de pipeline. Properties spreek je aan met de puntoperator. fileciteturn2file4L32-L35

Vergelijkingsoperatoren

-eq   gelijk
-ne   niet gelijk
-gt   groter dan
-lt   kleiner dan
-ge   groter of gelijk
-le   kleiner of gelijk
-like
-notlike
-match
-notmatch
-and
-or
-not

Deze operatoren worden in Where-Object gebruikt voor filtercriteria. fileciteturn2file4L33-L35

Filtervoorbeelden

Get-ChildItem .\Documents | Where-Object { $_.Attributes -eq "d" }

Get-Service | Where-Object {
  $_.Status -eq "Running" -and $_.CanPauseAndContinue -eq $true
}

Get-Process | Where-Object { $_.Name -like "*shell*" }

Die drie voorbeelden komen rechtstreeks uit de lesvoorbeelden. fileciteturn2file4L36-L38

Van terminal naar script

Een PowerShell-script is een verzameling commando’s in een tekstbestand met de extensie .ps1. Vanaf hier bouw je verder met variabelen, arrays, condities en lussen. Dat wordt in de tweede helft van de les aangekondigd. fileciteturn2file4L39-L40

# hello.ps1
$name = Read-Host "Wat is jouw naam?"
Write-Output "Hallo $name"
